# CCNA Labs

Hands-on networking labs created while preparing for the Cisco CCNA certification.

## Topics

- VLAN
- Inter-VLAN Routing
- Static Routing
- OSPF
- DHCP
- NAT/PAT
- ACL
- STP
- EtherChannel
- SSH
- IPv6
- OpenWrt Home Lab

## Tools

- Cisco Packet Tracer
- Cisco IOS
- OpenWrt
- Wireshark

## Goals

- Build practical networking skills
- Document lab configurations
- Demonstrate networking knowledge for Network Engineer positions
